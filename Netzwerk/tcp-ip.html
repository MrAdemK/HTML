TCP/IP

Bei TCP/IP handelt es sich um eine Gruppe von Protokollen, die die Grundlage für das Internet und andere Netzwerke bilden.

Dabei werden eigentlich sogar noch mehr Protokolle unter diesem Begriff zusammengefasst: Auch das Internet Control Message Protocol (ICMP) und das User Datagram Protocol (UDP) zählt man zu dieser Gruppe.
