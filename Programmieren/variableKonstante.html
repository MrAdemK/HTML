Unterschied Variable und Konstante

Man bezieht sich hierbei auf das Verhalten einer Zahl, wenn man das Funktionsargument verändert. Ist sie veränderlich, so nennt man sie Variable, bleibt sie gleich, heißt sie Konstante.

Beispiel für Konstante:

Sie finden viele Konstanten im Rahmen selbst, z.B.: in der Math-Klasse, wo eine Konstante für PI definiert wurde:

Console.WriteLine(Math.PI);

oder selber deklarieren:

const int TheAnswerToLife = 42;
