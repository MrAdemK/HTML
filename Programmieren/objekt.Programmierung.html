Objektorientierten Programmierung

  • Abstraktion: Das Schlüsselwort abstract ermöglicht die Erstellung von Klassen und Klassenmembern, die unvollständig sind und in einer abgeleiteten Klasse implementiert werden müssen. The abstract keyword enables you to create classes and class members that are incomplete and must be implemented in a derived class.
  • Klasse: Die Datenstruktur eines Objekts wird durch die Attribute (auch Eigenschaften) seiner Klassendefinition festgelegt. Das Verhalten des Objekts wird von den Methoden der Klasse bestimmt. Klassen können von anderen Klassen abgeleitet werden (Vererbung). Dabei erbt die Klasse die Datenstruktur (Attribute) und die Methoden von der vererbenden Klasse (Basisklasse).
  • Datenkapselung: Als Datenkapselung (englisch encapsulation, nach David Parnas auch bekannt als information hiding) bezeichnet man in der Programmierung das Verbergen von Daten oder Informationen vor dem Zugriff von außen.
  • Polymorphismus:
    • Statische Polymorphismus:
      • Methode wird überladen
      • MethodenName ist gleich, aber die Parameterlisten verschieden
    • dynamische Polymorphismus:
      • Methode wird überschrieben
      • Methode wurde von Superklasse geerbt
      • Gleiche Methodensignatur, Rumpf geändert
      • Erst zur Laufzeit wird entschieden, welche Methode aufgerufen wird
  • Vererbung: Vererbung heißt vereinfacht, dass eine abgeleitete Klasse die Methoden und Attribute der Basisklasse ebenfalls besitzt, also „erbt“. Somit kann die abgeleitete Klasse auch darauf zugreifen. Neue Arten von Objekten können auf der Basis bereits vorhandener Objektdefinitionen festgelegt werden. Es können neue Bestandteile hinzugenommen werden oder vorhandene überlagert werden.
