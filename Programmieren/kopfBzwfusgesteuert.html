"kopfgesteuert" bzw. "fußgesteuert" im Zusammenhang mit Schleifen

Eine kopfgesteuerte Schleife ist eine Kontrollstruktur. Sie bietet die Möglichkeit, Codezeilen zu wiederholen. ... In Java wird eine kopfgesteuerte Schleife mit dem Schlüsselwort while deklariert, weswegen sie oft auch als while-Schleife bezeichnet wird.

Fußgesteuerte Schleifen gehören zu den sogenannten Kontrollstrukturen. Eine fußgesteuerte Schleife ist einer kopfgesteuerten sehr ähnlich. Der Programmfluss durchläuft eine fußgesteuerte Schleife immer mindestens ein Mal, bevor getestet wird, ob zurück zum Kopf der Schleife gesprungen werden soll.
