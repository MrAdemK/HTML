Mail-Server

Ein Mail-Server ist eine Software, die Nachrichten und Mitteilungen

  • emfangen
  • senden
  • zwischenspeichern
  • und weiterleitet

Die Nachrichten können von E-Mail-Clients abgerufen und bearbeitet werden.

Die von einem E-Mail-Client an einen E-Mail-Server gesendeten E-Mails werden solange zwischengespeichert bis sie vom Empfänger abgeholt werden. Dies erfolgt mit dem Post Office Protocol (POP) oder dem Internet Message Access Protocol (IMAP).
